# simpLpdf

Minimal iPad PDF utility focused on:

* fast page selection
* simple export workflows
* custom gestures (2-finger undo, 3-finger redo)

## Tech

* Swift
* SwiftUI
* PDFKit
* PencilKit (later)

## Scope (v1)

* Import PDF
* View pages
* Select pages
* Export selected pages:

  * single combined PDF
  * individual PDFs

## Non-goals (v1)

* OCR
* text editing
* cloud sync
* form filling
* complex annotation systems

## Status

MVP in progress via Codex agent workflow.
